# Go和Vue.js环境搭建及简单应用开发教程
本文档将引导您如何在CentOS系统上安装Go、Node.js（使用NVM）、Vue CLI以及Gin框架，并创建一个简单的Vue前端和一个Gin后端应用。
## 1. 安装Go
### 1.1 安装Go语言环境
在命令行执行以下命令安装Go：
```bash
#添加EPEL仓库：
sudo yum install epel-release
#更新仓库信息
sudo yum update
#安装
sudo yum install golang
```
### 1.2 测试安装
安装完成后，验证Go版本：
```bash
go version
```
## 2. 安装Node.js环境（使用NVM）
### 2.1 安装NVM和Node.js
执行以下命令安装NVM，并使用NVM安装Node.js：
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
source ~/.bashrc
nvm install 16
nvm use 16
node -v
npm -v
```
### 2.2 安装jq
执行以下命令安装jq，一个轻量级的JSON处理器：
```bash
sudo yum install jq
jq --version
```
## 3. 创建Vue简单Web应用
### 3.1 创建项目目录
创建一个新目录，并初始化Vue项目：
```bash
mkdir application
cd application
mkdir web
cd web
npm init -y
npm install -g @vue/cli
vue create .
```
### 3.2 配置Vue项目
编辑`vue.config.js`文件，设置开发服务器端口为9528：
```javascript
const { defineConfig } = require('@vue/cli-service')

module.exports = defineConfig({
  transpileDependencies: true,
  devServer: {
    port: 9528, // 设置端口号为9528
  },
})
```
### 3.3 启动前端服务
在项目目录中，启动Vue开发服务器：
```bash
npm run serve
```
## 4. 创建Gin后端
### 4.1 创建后端项目目录
创建后端项目目录，并初始化Go模块：
```bash
mkdir backend
cd backend
go mod init my-gin-app
go get github.com/gin-gonic/gin
```
### 4.2 编写Gin后端代码
在`backend`目录中创建`main.go`文件，并编写以下代码：
```go
package main
import "github.com/gin-gonic/gin"
func main() {
    // 创建一个Gin实例
    r := gin.Default()
    // 定义一个简单的路由
    r.GET("/", func(c *gin.Context) {
        c.JSON(200, gin.H{
            "message": "Hello",
        })
    })
    // 监听并服务在9090端口
    r.Run(":9090")
}
```
### 4.3 运行Gin服务器
在`backend`目录中运行Gin服务器：
```bash
go run main.go
```
现在，您的Vue前端和Gin后端应用应该都运行在9090端口上。访问`http://localhost:9090/`将显示Vue前端页面，而访问`http://localhost:9090/`将返回Gin后端响应的"Hello"消息。



本教程介绍了如何在Linux环境下搭建Go和Node.js的开发环境，并创建一个简单的Vue.js前端应用及Go后端服务。
