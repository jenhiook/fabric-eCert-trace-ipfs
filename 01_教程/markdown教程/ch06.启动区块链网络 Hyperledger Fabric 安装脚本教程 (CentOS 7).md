# 启动区块链网络 Hyperledger Fabric 安装脚本教程 (CentOS 7)
## 环境要求
在开始安装之前，请确保您的系统满足[Hyperledger Fabric 2.5版本](https://hyperledger-fabric.readthedocs.io/en/release-2.5/prereqs.html)的安装要求。

官方具体教程：
https://hyperledger-fabric.readthedocs.io/en/release-2.5/install.html

## 安装 Fabric 和 Fabric 示例
1. **创建工作目录**：
   创建一个名为 `blockchain` 的目录并进入该目录。

   ```bash
   mkdir blockchain
   cd blockchain
   ```

2. **下载安装脚本**：
   使用cURL命令下载`install-fabric.sh`脚本，并为其赋予执行权限。

   下载 Fabric 示例、Docker 镜像和二进制文件

   ```bash
   curl -sSLO https://raw.githubusercontent.com/hyperledger/fabric/main/scripts/install-fabric.sh && chmod +x install-fabric.sh
   ```

3. **运行安装脚本**：
   使用`-h`选项查看脚本的可用选项。
   ```bash
   ./install-fabric.sh -h
   ```

4. **指定组件和版本**：
   使用`-f`或`--fabric-version`选项指定Fabric版本，使用`-c`或`--ca-version`选项指定Fabric CA版本。
   例如，要安装v2.5.9版本的Fabric和二进制文件，可以运行：
   
   ```bash
   ./install-fabric.sh --fabric-version 2.5.9 docker samples binary
   ```
   默认情况下，如果没有指定版本，则会安装最新版本的组件。
   
5. **安装组件**：
   指定要安装的组件，例如`docker`、`binary`和`samples`。
   例如，要同时安装Docker镜像、二进制文件和示例，可以运行：
   
   ```bash
   #如果执行上一步命令，该步可以省略
   ./install-fabric.sh docker samples binary
   ```
   如果未指定组件，则默认会安装所有组件。
   
6. **完成安装**：
   执行上述命令后，Fabric示例、Docker镜像和二进制文件将被下载并安装到您的系统中。

7. 查询镜像

   ```bash
   docker images
   ```
## 使用 Fabric 测试网络
官方文档提供了有关如何使用 Fabric 测试网络的详细指南。

官方教程：https://hyperledger-fabric.readthedocs.io/en/release-2.5/test_network.html

1. **进入测试网络目录**：
   ```bash
   cd fabric-samples/test-network
   ./network.sh -h
   ```
2. **启动网络**：
   使用以下命令来启动网络:
   ```bash
   ./network.sh up
   ```
3. **查看运行的 Docker 容器**：
   启动网络后，可以通过以下命令查看所有运行的 Docker 容器:
   ```bash
   docker ps -a
   ```
4. **停止网络**：
   当您完成测试或需要停止网络时，可以使用以下命令:
   ```bash
   ./network.sh down
   ```
## 更新PATH环境变量
为了方便使用，您可能需要将Fabric的bin、config目录添加到您的`PATH`环境变量中。

`fabric-samples` `/bin` and `/config` 目录中。这些二进制文件将帮助您与测试网络进行交互。

```bash
vim ~/.bashrc
export PATH=/root/05_soft/01_hyperledger-fabric/fabric-samples/bin:$PATH
source ~/.bashrc
```
