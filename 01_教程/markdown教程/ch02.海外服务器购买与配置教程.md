# ch02.海外服务器购买与配置教程
本文将指导您如何选择和购买海外服务器，并以ColoCrossing为例，详细说明配置过程。
## 一、海外服务器选择原因
1. **稳定性**：海外服务器VPS通常具有较高的稳定性，适合长期运行服务。
2. **便捷性**：可以直接通过Docker pull使用海外服务器，且访问GitHub速度较快。
3. **网络速度**：海外服务器网络速度快，有利于数据传输。
4. **软件下载**：如Go语言等软件可以直接下载，无需代理。
## 二、服务器选择
根据需求，选择2核2GB的海外服务器即可满足一般使用需求。
## 三、主机测评网页推荐
您可以通过主机测评网站 [IDCspy](https://www.idcspy.com) 查找和比较不同服务商的海外服务器。该网站提供了多种海外服务器的推荐与测评。
## 四、购买流程
推荐配置：2核CPU，2GB内存。

以下以ColoCrossing为例，说明购买流程。

### 1. 访问购买地址
打开 [ColoCrossing购买链接](https://go.idcspy.com/cloud.colocrossing.com/cart.php?a=add&pid=20)。
### 2. 使用优惠码
在结算时输入优惠码 `CLOUD5024` 以享受优惠。
### 3. 选择配置
- **操作系统**：选择 `CentOS 7.9 - 64bit`
- **数据中心位置**：选择 `纽约`
  填写您的邮件地址，并完成支付。
### 4. 获取SSH信息
支付成功后，您将收到服务器的SSH登录信息。
### 5. 使用ssh软件
例如FinalShell或者XTerminal，下载并安装 [FinalShell](http://www.hostbuf.com/)，使用SSH信息登录服务器。
## 五、服务器基本配置
登录成功后，修改SSH端口以提高安全性。

### 1. 修改SSH端口和密码

为了提高服务器安全性，建议修改默认SSH端口和密码。

```bash
# 登录服务器
# 修改SSH端口
sudo vi /etc/ssh/sshd_config
# 找到以下内容，修改Port后的端口号（例如：Port 220）
Port 22
# 保存并退出编辑器
sudo systemctl restart sshd
# 修改密码
sudo passwd
# 重启SSH服务
sudo systemctl restart sshd
```
**注意：**

密码也可以在官网的后台修改。

将ssh的22端口更换为220端口，请确保在执行前放行220端口

### 2. 重新登录

使用新的端口和密码，重新通过FinalShell软件登录服务器。

## 六、安装依赖环境

首先，确保您的包管理器是最新的：

```bash
sudo yum update  # 对于基于yum的系统
```

然后，安装所需的软件包。

```bash
sudo yum install -y git openssh-clients openssh-server make gcc gcc-c++ kernel-devel expect gawk texinfo openssl-devel bison flex fakeroot cmake unzip gperf autoconf dtc ncurses-devel pkgconfig bc python3 passwd openssh-clients vim file cpio rsync
```
