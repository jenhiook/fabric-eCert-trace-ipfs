# 1Panel 控制面板安装与基础配置教程
1Panel 是一款现代化的 Linux 服务器运维管理面板，开源且易于使用。本教程将指导您如何使用 1Panel 进行环境安装与管理，包括安装 Docker、配置防火墙、安装 MySQL 和 phpMyAdmin，以及创建数据库等操作。

## 一、安装 1Panel
官网：[https://1panel.cn/](https://1panel.cn/)

### 1. 使用 CentOS 在线安装命令

在 CentOS 服务器上，执行以下命令来安装 1Panel：
```bash
curl -sSL https://resource.fit2cloud.com/1panel/package/quick_start.sh -o quick_start.sh && sh quick_start.sh
```
此脚本将自动安装 1Panel 及其依赖的 Docker。

测试docker

```
docker version
```

### 2. 了解 1pctl 命令行工具
安装完成后，可以使用内置的命令行工具 `1pctl` 来进行运维管理。执行 `1pctl help` 查看相关命令说明。
## 二、1Panel 功能概述
### 1. 应用商店
应用商店支持应用的安装、升级、备份和恢复。您可以在这里安装 MySQL、phpMyAdmin、Go 等应用。
### 2. 网站管理
网站管理用于创建和管理 WEB 站点，依赖于 OpenResty 应用。
### 3. 数据库
可以创建和管理数据库，设置编码格式、密码和访问权限。
### 4. 容器
容器功能等价于 Docker 命令，可以查询容器状态、镜像、配置、日志等。

```bash
#显示本地存储库中的所有镜像
docker images
#列出当前系统上所有 Docker 容器
docker ps -a
```

### 5. 主机管理
主机管理包括**文件、终端、防火墙**等操作，类似于 Windows 文件夹和 SSH 客户端。

### 6. 工具箱
工具箱提供时区设置、进程守护（Supervisor）等功能。
### 7. 计划任务
用于管理定时执行的任务，如定期执行脚本、备份和访问 URL。
### 8. 日记审计
日记审计记录操作、登录和系统日志，可用于安全监控。
### 9. 高级功能
包括 WAF（Web 应用防火墙）和 1Panel 专业版监控功能。

### 10. 面板设置
支持面板的基础设置。
## 三、具体操作步骤
### 1. 配置防火墙
```bash
#安装 firewalld
sudo yum install firewalld
#启动 firewalld
sudo systemctl start firewalld
```

开放以下常用端口：

- HTTP: 80

- Flask: 5000

- MySQL: 3337

- phpMyAdmin: 18090

- Web 前端: 9528

- Web 后端: 9090

- 区块链浏览器: 8080
  在 1Panel 的主机下的防火墙设置中添加上述端口。

  也可以通过命名方式放行，例如放行23335

  ```bash
  #放行端口23335
  firewall-cmd --zone=public --add-port=23335/tcp --permanent
  #重新加载防火墙规则，使更改生效：
  firewall-cmd --reload
  ```

  
### 2. 安装 MySQL
在应用商店中安装 MySQL 8.2.0，并设置端口号为 3337。
### 3. 创建数据库
创建一个名为 `fabrictrace` 的数据库，允许外部访问，配置如下：
```plaintext
mysql:
  host: "127.0.0.1"
  port: 3337
  user: "sugar20240713"
  password: "sugar20240713"
  charset: "utf8mb4"
  db: "sugar20240713"
```
### 4. 安装 phpMyAdmin
在应用商店中安装 phpMyAdmin，并设置端口为 18090。
### 5. 安装 Go
在应用商店中安装 Go，并测试安装是否成功
```bash
go version
```
## 总结
通过以上步骤，您已经成功安装了 1Panel 控制面板，并配置了防火墙、MySQL、phpMyAdmin 和 Go 环境。

1Panel 提供了一个直观且功能强大的界面，使得 Linux 服务器的运维管理变得更加简单高效。
