# 使用VSCode远程连接服务器并查询Web项目
## 环境准备

确保您的计算机上已安装了Visual Studio Code（简称VSCode）编辑器。本教程将指导您如何使用VSCode的Remote Development扩展包连接到远程服务器，并在远程服务器上开发Vue.js项目。

## 1. 安装VSCode远程连接扩展

在VSCode中，为了能够远程连接服务器，首先需要安装相应的扩展。
### 1.1 打开扩展市场
点击VSCode左侧的扩展图标，进入扩展市场。
### 1.2 搜索并安装扩展
在搜索框中输入`ssh remote`，找到并安装以下扩展：

- **Remote - SSH**: 用于通过SSH连接到远程服务器。点击“Install”安装扩展。
  安装完成后，你将在VSCode界面看到`Remote Explorer`侧边栏。
## 2. 配置SSH连接
### 2.1 打开SSH配置文件
在VSCode中，点击`Remote Explorer`侧边栏，选择`SSH Targets`。然后右键点击，选择`Open SSH Config File`。
### 2.2 编辑SSH配置
在打开的`config`文件中，添加以下配置信息：
```plaintext
Host 127.0.0.1
  HostName 127.0.0.1
  User root
  Port 22
```
这里的`Host`是一个自定义的连接名称，`HostName`是服务器的IP地址，`User`是登录用户名，`Port`是SSH服务端口。
### 2.3 建立连接
保存`config`文件后，在`Remote Explorer`中点击刚才配置的`Host`名称旁边的`Connect`按钮。
### 2.4 选择连接路径
连接成功后，服务器上的文件系统将显示在`Remote Explorer`中。选择`/root`路径或其他你想工作的路径。
## 3. 打开并配置Vue项目
### 3.1 打开Vue项目
在远程服务器上，找到你的Vue项目文件夹，右键选择`Open Folder`，然后在VSCode中打开。
### 3.2 安装Vue扩展
由于你现在是在远程环境中，可能需要安装一些针对Vue开发的扩展，比如`Vetur`、`ESLint`、`Prettier`等。
在扩展市场中搜索并安装这些扩展，确保它们适用于远程开发环境。
## 总结
通过以上步骤，你已经成功使用VSCode远程连接到服务器，并打开了Vue项目。现在，你可以在本地环境中进行远程开发，享受与本地开发相同的编辑器功能。这种开发方式特别适合需要直接在服务器上进行开发和调试的场景。
