# 基于区块链技术的开源项目简易二次开发指南 —— 从农产品溯源到电子证照存储的应用转型
## 项目概述
本项目是基于区块链技术的开源项目，原先是用于农产品溯源的系统。本指南旨在指导如何将该系统进行业务领域的更换，转换为电子证照存储与溯源系统，并对其进行简易的二次开发。内容涵盖了前端界面的调整、后端功能的优化、以及必要的配置更新，以适应新的业务需求。通过本指南，开发者可以快速上手并实现系统的业务领域转换和功能扩展。

将原有属性：

- 种植户：农产品名称、产地、种植时间、采摘时间、种植户名称

- 运输司机：年龄、联系电话、车牌号、运输工具

- 个人用户：电子证照类型、身份证号、性别、联系电话、个人用户名称

- 技术支撑实体：实体名称、服务类型、安全认证等级、地址、联系方式
